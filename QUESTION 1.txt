Differences between Http/1.1 and Http/2

  Http/1.1 ;
1. HTTP stands for hypertext transfer protocol and it is used in client server communication..
2. HTTP/1.1 was created in the year 1997.
3. HTTP/1.1 works on the texual format.
4. There is head of line blocking thats blocks all the requests behind it until doesnt get its all resources.
5. HTTP/1.1 uses request resources inlining for use getting multiple pages.
6. HTTP/1.1 compresses data by itself.

  Http/2  ;
1. HTTP/2 was created in the year 2015.
2. HTTP/2 was developed over the SPDY protocol.
3. HTTP/2 works on the binary protocol.
4. HTTP/2 allows multiplexing so one TCP connection is required for multiple requests.
5. HTTP/2 uses PUSH frame by server that collects all multiple pages.
6. HTTP/2 uses HPACK for data compression.
